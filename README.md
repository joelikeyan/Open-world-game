# Open-world-game

This repository collects high-fidelity scene definitions for an open-world shooter sandbox. Each scene is stored as a standalone JSON document under `scenes/` and can be consumed by any runtime capable of parsing the `scene.create` contract.

## Available scenes

- `Cincinnati_Riverfront_FPS`: A photogrammetry-based recreation of Cincinnati's riverfront with dynamic weather, adaptive enemy AI, and free-roam or combat-focused objectives.
